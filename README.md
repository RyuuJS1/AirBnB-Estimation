# Sistema de estimación de precios de AirBnB

Un proyecto integral de aprendizaje automático centrado en el análisis de datos del mercado de Airbnb y la predicción de precios de propiedades según las características de los anuncios.

Desarrollado como un proyecto académico colaborativo, este sistema combina análisis estadístico de datos, modelos de aprendizaje automático y una plataforma de predicción web implementada en la nube. La aplicación permite a los usuarios estimar el valor de mercado aproximado de una propiedad de Airbnb introduciendo sus características y servicios.

# Resumen del proyecto

El objetivo de este proyecto fue explorar el comportamiento del mercado de Airbnb mediante técnicas de ciencia de datos y desarrollar un sistema predictivo capaz de estimar los precios de alquiler con alta precisión.

El proyecto incluyó:

- Limpieza y preprocesamiento de datos
- Análisis exploratorio de datos (AED)
- Análisis estadístico
- Ingeniería de características
- Entrenamiento y evaluación de modelos de aprendizaje automático
- Despliegue de una aplicación web interactiva

El resultado final es una plataforma en la nube donde los usuarios pueden estimar dinámicamente los precios de las propiedades de Airbnb utilizando modelos predictivos entrenados.

## 📊 Flujo de trabajo de aprendizaje automático

### 1. Limpieza de datos
El conjunto de datos se procesó para mejorar la consistencia y la fiabilidad mediante:
- Eliminación de duplicados
- Gestión de valores faltantes
- Corrección de datos inconsistentes
- Estandarización de formatos
- Filtrado de características irrelevantes

### 2. Análisis exploratorio de datos

Se realizaron análisis estadísticos y visualizaciones para identificar:
Patrones de distribución de precios
Correlación entre variables
Tendencias del mercado
Importancia de las características
Valores atípicos y anomalías

### 3. Ingeniería de características
Las características relevantes se transformaron y codificaron para mejorar el rendimiento del modelo.

### 4. Entrenamiento del modelo
Se entrenaron y compararon diferentes modelos de regresión para identificar el enfoque con mejor rendimiento para la predicción de precios.

### 5. Evaluación del modelo
Los modelos se evaluaron utilizando métricas de regresión estándar como:
- MAE (Error absoluto medio)
- RMSE (Error cuadrático medio de la raíz)
- R²

---

## 🌐 Aplicación Web

El proyecto incluye una aplicación web interactiva que permite a los usuarios estimar los precios de los alojamientos de Airbnb a partir de datos personalizados como:
- Tipo de alojamiento
- Número de habitaciones
- Ubicación
- Servicios
- Capacidad
- Otras características del alojamiento
El modelo de aprendizaje automático entrenado procesa la información y devuelve un precio de mercado estimado.
